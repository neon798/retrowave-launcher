# RetroWave Launcher

[![Build](https://github.com/neon798/retrowave-launcher/actions/workflows/build.yml/badge.svg)](https://github.com/neon798/retrowave-launcher/actions/workflows/build.yml)

A synthwave-themed Android launcher based on [Lawnchair 16](https://github.com/LawnchairLauncher/lawnchair).

## Features

- Synthwave neon color palette (purple, pink, orange, blue accents)
- Semi-transparent app drawer and hotseat backgrounds
- All Lawnchair 16 features: Material 3 theming, At a Glance, QuickSwitch, icon packs, etc.

## Download

Download the latest APK from [GitHub Actions](https://github.com/neon798/retrowave-launcher/actions) — select the latest run and download the `apk` artifact.

## Building

```bash
./gradlew assembleLawnWithQuickstepGithubDebug
```

Requires JDK 21 and Android SDK 35.
